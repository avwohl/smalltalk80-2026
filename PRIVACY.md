# Privacy Policy

**st80-2026** is an open source Smalltalk-80 virtual machine implementing
the 1983 Xerox Blue Book image. It runs on macOS, iOS, Windows, and Linux.

## Data Collection

This app does **not** collect, store, or transmit any personal data.

## Network Usage

The app itself makes no network connections. The Smalltalk-80 image that
runs inside the VM is the 1983 Blue Book image and does not initiate
network activity on its own.

## Local Storage

All data (the Smalltalk image, changes file, sources file, and any
snapshots you save) is stored locally on your device in the app's
private container. This data is not accessible to other apps and is
not transmitted anywhere.

## Analytics

This app does not include any analytics, tracking, or advertising
frameworks.

## Third-Party Services

This app does not use any third-party services that collect user data.

## Contact

If you have questions about this privacy policy, please open an issue at:
https://github.com/avwohl/smalltalk80-2026/issues

## Changes

This privacy policy may be updated occasionally. Changes will be posted
to this repository.
